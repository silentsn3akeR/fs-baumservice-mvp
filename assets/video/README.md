# Freigegebene Videos

Hier kurze, freigegebene Clips fuer die Website ablegen.

Empfehlung:

- Hochformat-Clips aus Instagram/Reels nur verwenden, wenn der Inhaber die Originaldatei freigibt.
- Kurze Clips auf 8 bis 18 Sekunden schneiden.
- Ohne Ton planen oder mit optionalem Ton.
- Als `.mp4` exportieren.

Empfohlene Dateinamen:

- `baumfaellung-kontrolliertes-abtragen.mp4`
- `wurzelstockfraesen-einsatz.mp4`
- `baumpflege-seilklettertechnik.mp4`
